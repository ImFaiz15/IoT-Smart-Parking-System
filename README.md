# IoT Smart Parking System 🚗

## Overview
This project is an IoT and Web-Based Smart Parking System designed to monitor parking slot availability in real time using ultrasonic sensors and ESP8266.

## Features
- Real-time slot monitoring
- Web dashboard display
- Automatic vehicle detection
- WiFi-based communication

## Technologies Used
Hardware:
- Ultrasonic Sensor
- NodeMCU ESP8266

Software:
- Arduino IDE
- HTML
- CSS
- JavaScript
- Firebase

## Working
Sensors detect vehicle presence and send data through ESP8266 to the cloud database. The web dashboard updates parking availability instantly.

## Applications
- Smart Cities
- Shopping Malls
- Hospitals
- College Campuses
